## Overview

This is a personal portfolio website created to showcase my projects, skills, and experience. The site includes multiple interactive elements and is structured for easy updates and scalability.

### Features
- **Responsive Design**: Optimized for various devices, ensuring a seamless experience on desktops, tablets, and smartphones.
- **Interactive Background**: Features particle.js animations for enhanced visual appeal.
- **Profile Section**: Displaying personal information, projects, and skills.
- **Smooth Scrolling**: Navigation is enhanced by smooth scrolling and easy-to-use links.

### Technologies Used:
- HTML5
- CSS3 & SCSS
- JavaScript (particles.js, custom scripts)
- Bower for package management
